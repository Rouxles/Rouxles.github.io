LSE Progression

This is the order of a functional path for getting better at LSE. Learn the next one when you feel you're ready to - no need to wait for sub-x.

  • Learn the EO Flowchart - they contain optimal MU solutions for each EO case
  • Start tracking LR edges while doing EO, and try to force one LR edge to be on the D layer after you finish EO, and use that to start doing 4a -> 4b pauseless
  • Learn DFDB recognition and get used to it in order to have pauseless 4b -> 4c transitions
  • Learn EOLR (Don't avoid misoriented centers)
